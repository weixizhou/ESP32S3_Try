

# 简介
工程主要为esp32S3在Arduino平台上的学习与验证<br>
将不涉及任何商业用途.

# 开发环境
- Windows/linux
- Vscode+PlatformIO

# 工程目录
```
.
├── build [编译输出文件]
│   ├── esp32dev
│   │   └── idedata.json
│   └── project.checksum
├── include  [头文件]
│   ├── Config.hpp
│   ├── main.hpp
│   └── README
├── lib  [第三方库]
│   └── README
├── platformio.ini [platformio工程配置文件(一般不动)]
├── README.md
├── src  [程序源文件]
│   ├── app_ctrl.cpp
│   ├── app_home.cpp
│   ├── app_main.cpp
│   ├── function
│   │   └── function_dev_1.cpp
│   └── main.cpp
└── test  [测试文件]
    └── README
```
